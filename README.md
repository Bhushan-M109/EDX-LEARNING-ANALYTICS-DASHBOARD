# EdX Learning Analytics Dashboard

## Overview
This project analyzes edX academic course data to understand learner behavior, certification trends, engagement levels, and demographics.

## Dataset
Contains 50,000+ learner records with attributes such as:
- Country
- Age
- Gender
- Certification status
- Course participation
- Activity metrics

## Key Insights
- Certification distribution across regions
- Learner demographic trends
- Engagement vs completion patterns
- Course popularity analysis

## Tools Used
- Tableau
- Data Cleaning
- Data Visualization

## Files
- Edx Academic Business Report.twbx → Main dashboard
- Edx Academic Business Report - Summary.twbx → Summary dashboard
- Dataset CSV → Raw data

## Use Case
This dashboard helps educators and platforms:
- Understand learner engagement
- Improve course completion rates
- Identify target demographics
- Optimize course design
